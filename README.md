# EEVE 타로점 게임 기획서

## 1. 프로젝트 개요

### 1.1 프로젝트 명
**"EEVE Tarot - AI 타로 리더"**

### 1.2 목적
EEVE 한국어 AI 모델을 활용하여 사용자에게 타로 카드 해석 서비스를 제공하는 대화형 타로점 게임 개발

### 1.3 타겟 사용자
- 타로에 관심 있는 20-40대
- AI 기술에 호기심이 있는 사용자
- 일상적인 고민 상담이 필요한 사람들

### 1.4 핵심 가치
- **개인화**: 각 사용자의 상황에 맞춘 맞춤형 해석
- **접근성**: 언제 어디서나 타로 상담 가능
- **몰입감**: 신비로운 타로 경험 제공
- **한국어 특화**: EEVE 모델의 자연스러운 한국어 표현

## 3. 게임 플로우

### 3.1 전체 흐름도
```
[시작] 
  ↓
[질문 입력] - 사용자가 고민/질문 작성
  ↓
[스프레드 선택] - 1장/3장/켈틱 크로스 등
  ↓
[카드 셔플 & 선택] - 애니메이션과 함께 카드 뽑기
  ↓
[카드 공개] - 뽑힌 카드 이미지 표시
  ↓
[AI 해석] - EEVE 모델이 카드 해석
  ↓
[추가 질문/저장] - 대화 계속 또는 결과 저장
  ↓
[종료]
```

### 3.2 상세 단계

#### STEP 1: 웰컴 화면
- 신비로운 UI/UX (보라색/금색 테마)
- 짧은 인트로 애니메이션
- "어떤 것이 궁금하신가요?" 메시지

#### STEP 2: 질문 입력
```
입력 예시:
- "나의 연애운이 궁금해"
- "이직을 해야 할까?"
- "이번 주 운세를 알려줘"
```
- 자유 텍스트 입력
- 선택형 질문 카테고리 제공
  - 💕 연애/관계
  - 💼 직업/진로
  - 💰 재물/금전
  - 🏥 건강
  - 🎯 일반 운세

#### STEP 3: 스프레드 선택
1. **원 카드 (One Card)**
   - 가장 간단한 형태
   - 오늘의 운세, 간단한 예/아니오 질문

2. **쓰리 카드 (Three Card Spread)**
   - 과거 - 현재 - 미래
   - 또는 상황 - 행동 - 결과

3. **켈틱 크로스 (Celtic Cross)** [고급]
   - 10장의 카드
   - 복잡한 상황 분석

#### STEP 4: 카드 셔플 & 선택
- 78장의 타로 카드 덱 애니메이션
- 사용자가 직접 카드 클릭하여 선택
- 카드 뒤집기 애니메이션

#### STEP 5: AI 해석
EEVE 모델이 다음을 생성:
1. **카드 기본 의미** (간단 설명)
2. **현재 상황 해석** (사용자 질문 연결)
3. **조언/메시지** (구체적 행동 제안)
4. **키워드** (핵심 단어 3-5개)

#### STEP 6: 추가 대화
- "더 궁금한 게 있으신가요?"
- 사용자 추가 질문 가능
- RAG 방식으로 이전 해석 참조

## 5. EEVE 프롬프트 설계

### 5.1 시스템 프롬프트
```
당신은 30년 경력의 전문 타로 리더입니다. 
사용자의 질문과 뽑힌 타로 카드를 바탕으로 
깊이 있고 공감적인 해석을 제공합니다.

해석 시 다음을 포함하세요:
1. 카드의 기본 의미
2. 사용자 상황과의 연결
3. 구체적이고 실천 가능한 조언
4. 희망적이면서도 현실적인 톤

주의사항:
- 단정적 예언은 피하고 "가능성"으로 표현
- 부정적 카드도 긍정적 관점 제시
- 한국 문화에 맞는 표현 사용
```

### 5.2 입력 프롬프트 예시
```
사용자 질문: "이직을 해야 할까요?"

뽑힌 카드:
1. 과거 위치: The Devil (정방향)
   - 의미: 속박, 중독, 물질적 집착
2. 현재 위치: Eight of Pentacles (역방향)
   - 의미: 무기력, 실력 정체, 동기 부족
3. 미래 위치: The Sun (정방향)
   - 의미: 성공, 기쁨, 긍정적 변화

위 카드들을 바탕으로 사용자의 이직 고민에 대해 
과거-현재-미래 흐름으로 해석해주세요.
```

```python
# 양자화로 추론 속도 개선
from transformers import AutoModelForCausalLM, AutoTokenizer
import torch

model = AutoModelForCausalLM.from_pretrained(
    "yanolja/EEVE-Korean-10.8B-v1.0",
    device_map="auto",
    torch_dtype=torch.float16,  # 메모리 절약
    load_in_8bit=True  # 8bit 양자화
)
```
